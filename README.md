# 🚀 LeetTrack

LeetTrack is a full-stack LeetCode progress tracker built with Spring Boot.

## Features

- JWT Authentication
- User Registration & Login
- Add solved problems
- Update/Delete problems
- Favorite problems
- Search by title
- Filter by difficulty
- Search by date
- Current & longest streak
- Monthly activity
- Topic-wise statistics
- Difficulty distribution
- Dashboard API
- Swagger Documentation
- Docker support

## Tech Stack

### Backend
- Java 17
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- Hibernate
- MySQL
- Maven

### Documentation
- Swagger OpenAPI

### Testing
- JUnit 5
- Mockito
- MockMvc

### DevOps
- Docker
- Docker Compose

## Running Locally

Clone the repository

```bash
git clone <your-repository-url>
```

Build the project

```bash
./mvnw clean package
```

Run using Docker

```bash
docker compose up --build
```

Swagger

```
http://localhost:8080/swagger-ui/index.html
```
