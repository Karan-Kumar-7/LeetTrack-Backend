# 📖 Overview

LeetTrack is a **full-stack web application** that helps programmers organize and analyze their LeetCode journey.

Instead of simply keeping track of solved problems, LeetTrack provides a complete dashboard with analytics, progress visualization, notes, complexity tracking, favorites, exports, and productivity features.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/LeetTrack.git
```

---

## Backend

```bash
cd backend
```

Configure

```properties
application.properties
```

Run

```bash
mvn spring-boot:run
```

---

## Frontend

```bash
cd frontend
```

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```

---
